Solufa

Solufaは"Mithril.js"で記述するWeb3D/VRライブラリです。
Mithril.jsを扱えることを前提としています。
Mithril.jsの詳細は公式サイトをご覧ください。

Global Object

window.Solufa === window.S; // true
"S"が他のライブラリと競合する場合はnoConflictを呼んでください。 
Solufa.noConflict();
// window.S !== window.Solufa;

Virtual DOM

Mithril.jsとは競合しません。Solufaの名前空間に"m"関数が定義されています。 

// window.m === undefined;
var m = Solufa.m;
変数"m"が定義されているスコープ内でのみMSXTransformerを利用できます。 
m.render( S.document.head, <rdr init={{ frame: "#solufa", antialias: true }}>
  <vp cam="#cam"/>
</rdr> );
JavaScript 
var myColor = "#f00";
m.render( S.document.body, m( "scene", [
  m( MyComponent, { color: myColor } )
]) );

DOMContentLoaded

S( function( m ) {
  // m === Solufa.m;
}, function() {
  // disabled WebGL handling
});
