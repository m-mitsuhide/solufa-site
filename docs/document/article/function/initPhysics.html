S.initPhysics( url );

Call before to set the "physics" attributes of the element.
Only <mesh /> is able to set the "physics" attributes.
You can change the setting of the physical world in the "physicsWorld" attribute of <scene />.

// To specify the URL of the cannon.js.
S.initPhysics( "https://cdnjs.cloudflare.com/ajax/libs/cannon.js/0.6.2/cannon.min.js" );
S( function( m ) {
  // enabled physics
});
